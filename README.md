# RCE_GIS_Intro
This repository is meant to share instructions, slides and data for interns at the Dutch Heritage Agency (Rijksdienst voor het Cultureel Erfgoed). 


## Learning objectives
After this workshop the participant will:
- Have a basic understanding of different types of spatial data (both raster and vector data) and projected coordinate systems.
- Have a basic understanding on the type of spatial analyses that can be performed using Geographic Information System software
- Be able to combine and compare maps from different sources using Geographic Information System software.
- Be able to understand differences in data types

**Specifically the participant will be trained to use the Free and Open Source Software QGIS.**   
- Create, edit and query vector (points, lines and polygons) data
- Calculate areas, distances and perform basic descriptive statistics
- Compare different datalayers with each other (by changing the transparency)
- Generate thematic maps with vector data
- Transform data to a different projected coordinate system
- Load data from different data portals i.e. [Erfgoedatlas](https://rce.webgis.nl/nl/map/erfgoedatlas), [PDOK](https://www.pdok.nl/), [OpenStreetmap](https://www.openstreetmap.org/)
- Georeference historical maps and or historical aerial photographs

For this workshop the participants are asked to install the 3.44 LTR version of QGIS on their local machine.
[Download QGIS](https://download.qgis.org/downloads/QGIS-OSGeo4W-3.44.9-1.msi) (this will work both on MS and Apple)
